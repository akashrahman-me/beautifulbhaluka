## Interaction Flow

1. **Main Screen**:
    - User scrolls vertically to see different categories
    - User swipes horizontally within each category to see products
    - User clicks "à¦†à¦°à¦“ à¦¦à§‡à¦–à§à¦¨" (See More) to view all products in category

2. **Category Screen**:
    - Shows all products in grid layout (2 columns)
    - User can sort products
    - User clicks back button to return to main screen
    - User clicks product to see details
